## go test

1. Go는 간편하게 사용할 수 있는 테스트 프레임워크를 내장하고 있는데, "go test" 명령을 실행하여 테스트 코드들을 실행할 수 있다. 
2. "go test"는 현재 폴더에 있는 *_test.go 파일들을 테스트 코드로 인식하고, 이들을 일괄적으로 실행한다.
